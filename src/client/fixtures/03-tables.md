# API Surface

| --------------- | :----: | --------- | ---------------------- |
| `/login`        |  POST  | none      | issues a session token |
| `/refresh`      |  POST  | refresh   | rotates the token      |
| `/logout`       |  POST  | session   | revokes the token      |
